Prismless
=========
An online forum builder that makes it easy to set up a public or private forum with subforums.
It does not collect any data for surveillance purposes.

How to Use
==========
* Clone the repo
* `bundle install`
* `rake db:migrate`
* `rails s`
* Go to localhost:3000 in a browser
